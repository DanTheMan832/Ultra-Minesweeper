HOW TO PLAY

Minesweeper is a strategy game about clearing an area whilst avoiding mines. The game is made up of a grid of boxes, each of which has a chance to contain a mine. You win the game by revealing all boxes which don't contain mines. You can reveal any boxes you choose and use the information this gives you to avoid any mines. If a box doesn't contain a mine then it will contain a number which represents the total number of mines in all surrounding boxes (including diagonals).

If you think that a box contains a mine then you can place a flag to remind yourself to not reveal that box. If you change your mind then you can always remove the flag in the future.

CONTROLS


Keyboard and Mouse

Left click a box to reveal it.
Middle click or press the spacebar whilst hovering over a box to place or remove a flag.
Press R to restart

TOUCHSCREEN

Tap the toggle mode button to switch between reveal mode and flag mode.
In reveal mode tapping on a box will reveal it.
In flag mode tapping on a box will add or remove a flag.

User Interface

  • 1) Difficulty select menu. If you select a new difficulty it will restart the game.
  • 2) Displays the current difficulty.
  • 3) Toggle Mode button. Allows you to switch between reveal mode and flag mode. Only works on touchscreens. (See Controls)
  • 4) Displays the current mode you are in - Reveal mode or Flag mode. (See controls)
  • 5) Restart Button - resets the game and sets the difficulty to the one currently selected.
  • 6) Timer
  • 7) The total number of mines hidden in the grid
  • 8) The total number of flags you have placed
  • 9) Menu - Use this to navigate the site.
